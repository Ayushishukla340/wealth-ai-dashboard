# wealth-ai-dashboard

# 💸 PulseWealth AI – Smart Investment Co-Pilot

PulseWealth AI is a futuristic landing page concept for an AI-powered finance platform that helps users grow their wealth through intelligent automation and market insights.

---

## 🚀 Features

- 🌌 Futuristic UI with glowing gradients  
- 📊 Dashboard-style interface  
- 📈 Simulated portfolio growth (dynamic)  
- 🤖 AI-based buy/sell/hold signals  
- ⚡ Interactive CTA button  
- 🎨 Clean and modern design  

---

## 🧩 How It Works

1. User lands on the homepage  
2. Sees AI-powered finance concept and dashboard  
3. Clicks on **"See Your AI Portfolio Grow"**  
4. Portfolio values dynamically increase  
5. AI signals change in real-time (BUY / SELL / HOLD)

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  

---

## 🎯 Purpose

This project demonstrates a concept of an AI-driven financial assistant that:
- Analyzes user spending  
- Predicts market trends  
- Automates micro-investments  

The goal is to visually simulate how AI can manage and grow a user's investment portfolio.

---

## ⚠️ Note

This is a frontend prototype.  
All data and AI behavior are simulated using JavaScript for demonstration purposes.
